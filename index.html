When you start building web applications and APIs you often need something that interacts with databases for you. Maintaining relationships between tables, fetching related records and managing transactions gets very repetitive over time when using just raw queries. Sequelize abstracts these tasks away for you and provides you with a database layer. Sequelize is an ORM which is designed to work with Node.JS and IO.JS. It's written in pure JavaScript and supports many major database engines like MySQL, Postgres, SQLite and MSSQL.
